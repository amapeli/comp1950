The aim of this project is to redesign the current COMP 1950 with the following parameters...

  1. To improve access to information on the website (as per students and owners recommendations):
    • Assignment, exercises all in one easy place
    • Easy access to examples, references.
    • There is an area for latest news/updates/changes etc.
    • The lecture pages have lists of all subjects and a direct link to the information ie SEO Optimization, media queries. This will reduce time searching for certain subjects.
  2. Improve the layout style, with techniques learnt within the course:
    • Different media queries.
    • HTML5 coding.
    • Different style sheets.
    • Responsive Web design.
    • Different style sheets imported into the CSS.
    • CSS Descendant, Child and Attribute Selectors.
    • Server Side includes.
    • Tumblr blog.
    • This design is to look modern, sleek and easy to navigate.
  3. The function of the website is to complement the teachings within the class:
    • As this website is targeting current students, it is in the best interest of the lecturer time and resources to NOT concentrate on Alumni access and students, as the content is very closely related to the teachings, and may be taken out of context. Having an alumni page, will increase the work for the tutor and would need to be constantly updated.
    • The lecture notes in week order and in-class chronology. The lecture is to be downloaded separately each week.
